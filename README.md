# YaTube

## Описание:
Социальная сеть

## Технологии:

 - Python 3.7 
 - Django 2.2.19

## Запуск проекта в dev:

 1. Активировать venv
 2. Установите зависимости из файла `requirements.txt`:
`pip install -r requirements.txt`
 3. В папке с `manage.py` выполнить: 
`python3 manage.py runserver`
